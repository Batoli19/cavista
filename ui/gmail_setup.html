Gmail Setup Guide

  1. Install dependencies: pip install google-api-python-client google-auth-httplib2 google-auth-oauthlib
  2. Create OAuth Desktop credentials in Google Cloud.
  3. Save the downloaded file as credentials.json in the project root.
  4. Run a Gmail command once (for example: summarize the last email) and complete OAuth consent in your browser.
  5. Confirm token.json is created in the project root.

After setup, retry your Gmail command in ProjectForge.
